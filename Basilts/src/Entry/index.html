Basil Viewer Test on OSGrid

This page has you log into an OSGrid account and then view the OSGrid region BasilAlpha using the Basil browser-based viewer.

Be patient! When you login, you will see a horizon line. Wait. It will take a while for the world to load so don't get impatient. Eventually you will see the back of a duck and then the world load around the avatar.

Movement is via keyboard arrow keys or W-A-S-D keys. There is also the ALT-key/click orbit view (ESC to exit).

This is a VERY, VERY early, alpha quality version of the Basil viewer code. You and everyone else's avatar is a duck. You can move around. That's about all there is at the moment.

BasilAlpha is a regular OSGrid region so avatars can teleport and hypergrid to it. All avatars, no matter where they are from, will appear as ducks.

The odd "Grid login URL" is because the browser requires the login XMLRPC call to be done over a HTTPS connection. OSGrid does not use connection certificates so a reverse proxy has been setup at https://basil.bluestuff.org that proxies a request from https://basil.bluestuff.org/osgrid => http://osgrid.org/.

Privacy Policy: no personally identifying data is collected. The entered password is salted and hashed and sent through the above mentioned reverse proxy for login to OSGrid and is not stored or saved. Any information collected is the usual OpenSimulator session and connection information.
